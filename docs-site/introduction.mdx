## Why Overseer?

<AccordionGroup>
  <Accordion title="True Agentic Behavior">
    Powered by Vercel AI SDK's Tool Loop for complex, multi-step autonomous
    reasoning. Not just a chatbot — a real agent that can plan and execute
    multi-step tasks.
  </Accordion>
  <Accordion title="20+ LLM Providers">
    OpenAI, Anthropic, Google Gemini, Groq, Ollama, Azure, AWS Bedrock, Mistral,
    xAI Grok, and many more. Switch providers on the fly.
  </Accordion>
  <Accordion title="Multi-Platform Chat">
    Talk to your server via Telegram, Discord, or the built-in web interface.

  <Accordion title="Enterprise Security">

## Quick Install

```bash
curl -fsSL https://overseer.sh/install | bash
```

Or clone manually:

```bash
git clone https://github.com/ErzenXz/overseer.git
cd overseer
pnpm install
pnpm setup
pnpm dev
```
