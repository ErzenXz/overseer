# Quick Start

Get your self-hosted AI agent up and running in minutes.

## Prerequisites

<CardGroup cols={3}>
  <Card title="Node.js 20+" icon="node-js">
    [Download Node.js](https://nodejs.org)
  </Card>
  <Card title="pnpm" icon="package">
    `npm install -g pnpm`
  </Card>
  <Card title="Bot Token" icon="robot">
    From [@BotFather](https://t.me/BotFather) or Discord Developer Portal
  </Card>
</CardGroup>

## Installation

<Tabs>
  <Tab title="One-Line Install">
    ```bash
    curl -fsSL https://overseer.sh/install | bash
    ```
    This will clone the repo, install dependencies, and guide you through configuration.
  </Tab>
  <Tab title="Manual Install">
    ```bash
    # Clone the repository
    git clone https://github.com/ErzenXz/overseer.git
    cd overseer

    # Install dependencies
    pnpm install

    # Configure environment
    cp .env.example .env
    nano .env

    # Initialize database
    pnpm run db:init
    ```

  </Tab>
  <Tab title="Docker">
    ```bash
    git clone https://github.com/ErzenXz/overseer.git
    cd overseer

    # Development
    docker compose -f docker-compose.dev.yml up

    # Production
    docker compose up -d
    ```

  </Tab>
</Tabs>

## Configuration

Edit your `.env` file with the required settings:

```env
# Admin Credentials
ADMIN_USERNAME=admin
ADMIN_PASSWORD=your-secure-password

# Security Keys (generate with: openssl rand -hex 32)
ENCRYPTION_KEY=your-64-char-hex-key
SESSION_SECRET=your-session-secret

# Telegram Bot (optional)
TELEGRAM_BOT_TOKEN=123456:ABC-DEF...
TELEGRAM_ALLOWED_USERS=123456789

# Discord Bot (optional)
DISCORD_BOT_TOKEN=your-discord-token
DISCORD_ALLOWED_GUILDS=server-id

# Default LLM Provider
DEFAULT_PROVIDER=openai
DEFAULT_MODEL=gpt-4o
OPENAI_API_KEY=sk-...
```

## Start Services

```bash
# Web Dashboard (http://localhost:3000)
pnpm dev

# Telegram + Discord bots (separate terminal)
pnpm bots:dev
```

## First Steps

<Steps>
  <Step title="Open Web Dashboard">
    Navigate to [http://localhost:3000](http://localhost:3000) and login with
    your admin credentials.
  </Step>
  <Step title="Configure LLM Provider">
    Go to **Settings → Providers** and add at least one LLM provider (OpenAI,
    Anthropic, etc.)
  </Step>
  <Step title="Connect Chat Platform">
    Configure Telegram or Discord in **Settings → Interfaces**
  </Step>
  <Step title="Start Chatting">
    Message your bot and start managing your server with natural language!
  </Step>
</Steps>

<Tip>
  You can also use the built-in web chat in the dashboard without configuring
  Telegram or Discord.
</Tip>
