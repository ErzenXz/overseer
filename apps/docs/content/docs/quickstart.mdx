# Quick Start

This page is for people who want the shortest path from “nothing installed” to “I can chat with Overseer”.

If you are not technical, start here.

## What you are setting up

By the end of this guide, you will have:

- the Overseer web dashboard running
- an admin login you can use in the browser
- at least one AI provider connected
- optional Telegram or Discord chat later if you want it

## Before You Begin

You do **not** need Telegram or Discord to try Overseer.

The web chat works on its own. Bots are optional.

## Installation

<Tabs>
  <Tab title="One-Line Install">
    ```bash
    curl -fsSL https://overseer.sh/install | bash
    ```
    This is the easiest option. It clones the project, installs dependencies,
    and walks you through setup.
  </Tab>
  <Tab title="Manual Install">
    ```bash
    # Clone the repository
    git clone https://github.com/Quad-Labs-LLC/overseer.git
    cd overseer

    # Install dependencies
    pnpm install

    # Configure environment
    cp .env.example .env
    nano .env

    # Initialize database
    pnpm run db:init
    ```

  </Tab>
  <Tab title="Docker">
    ```bash
    git clone https://github.com/Quad-Labs-LLC/overseer.git
    cd overseer

    # Development
    docker compose -f docker-compose.dev.yml up

    # Production
    docker compose up -d
    ```

  </Tab>
</Tabs>

## Minimum Configuration

If you want the smallest possible setup, fill in these first:

```bash
# Admin Credentials
ADMIN_USERNAME=admin
ADMIN_PASSWORD=your-secure-password

# Security Keys (generate with: openssl rand -hex 32)
ENCRYPTION_KEY=your-64-char-hex-key
SESSION_SECRET=your-session-secret

# Telegram Bot (optional)
TELEGRAM_BOT_TOKEN=123456:ABC-DEF...
TELEGRAM_ALLOWED_USERS=123456789

# Discord Bot (optional)
DISCORD_BOT_TOKEN=your-discord-token
DISCORD_ALLOWED_GUILDS=server-id

# Default LLM Provider
DEFAULT_PROVIDER=openai
DEFAULT_MODEL=gpt-4o
OPENAI_API_KEY=sk-...
```

You can add Telegram and Discord later. They are optional.

## Start Services

```bash
# Web Dashboard (http://localhost:3000)
pnpm dev

# Telegram + Discord bots (separate terminal)
pnpm bots:dev
```

## Open the App

Open `http://localhost:3000` in your browser and log in with your admin username and password.

## First Things To Click

<Steps>
  <Step title="Log in">
    Open [http://localhost:3000](http://localhost:3000) and sign in with the
    admin username and password you set.
  </Step>
  <Step title="Add an AI provider">
    Go to **Settings → Providers** and add at least one provider such as OpenAI,
    Anthropic, or Google.
  </Step>
  <Step title="Test the web chat">
    Go back to the main chat screen and send a simple message like “What can you
    do?” or “Explain this server.”
  </Step>
  <Step title="Add bots later if you want">
    Telegram and Discord can be configured later in **Settings → Interfaces**.
  </Step>
</Steps>
